\section{Descripción general}

Este repositorio contiene el desarrollo completo de la tesis de maestría en Ingeniería Agroindustrial, enfocada en el diseño, modelado y escalamiento de una tecnología innovadora de secado de café de especialidad.

El proyecto propone un sistema de secado asistido por bomba de calor con control de \textbf{humedad relativa}, en lugar del enfoque tradicional basado en temperatura, con el objetivo de mejorar la calidad, eficiencia y reproducibilidad del proceso.

Trabajo desarrollado en la Universidad Nacional de Colombia – Sede Palmira, en colaboración con la Universidad del Valle.

\section{Problema}

El secado es una de las etapas más críticas en la poscosecha del café. Actualmente:

\begin{itemize}[leftmargin=*]
\item Existe alta dependencia de condiciones climáticas
\item Alta variabilidad en la calidad final
\item Dificultad para estandarizar procesos como \textit{honey} y \textit{natural}
\item Pérdidas económicas por defectos
\end{itemize}

\section{Propuesta de solución}

Se plantea el desarrollo de un sistema de secado basado en:

\begin{itemize}[leftmargin=*]
\item Bomba de calor
\item Control de humedad relativa del aire
\item Modelamiento de cinética de secado
\item Evaluación energética y de calidad sensorial
\end{itemize}

Esto permite:

\begin{itemize}[leftmargin=*]
\item Mayor control del proceso
\item Secado uniforme
\item Reducción de defectos
\item Preservación de compuestos clave del café
\end{itemize}
